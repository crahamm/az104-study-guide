1. Which of the following is a necessary condition when defining a security rule to allow inbound traffic for an application hosted on Azure?

- The source should be set to Any or a specific IP address range, and the action should be set to allow.

2. What happens if a new security rule is created with a priority value that is higher than existing rules?

- The new rule is processed after existing rules with lower priority values.

3. What scenario best describes an appropriate use of network security groups in Azure?

- When you need to restrict traffic flow to resources in your virtual network based on specific security rules.

4. Your organization needs to secure a new virtual machine in Azure that hosts sensitive data. Which configuration should you apply to the network security group to ensure only HTTPS traffic is allowed inbound while all other traffic is denied?

- Create a network security group rule allowing inbound traffic on port 443 and denying all other inbound traffic.

5. In a scenario where you need to manage security for different workloads with minimal complexity, which Azure service would allow you to create logical groupings of virtual machines and apply security rules efficiently?

- Application Security Groups

6. In a code review, you notice that virtual machines in a project are being organized to manage network traffic flow. The goal is to simplify security rule management based on workloads. What Azure feature should be used to achieve this?

- Application Security Groups

7. What is a key benefit of using application security groups in Azure?

- They allow you to define network security based on application logic rather than IP addresses.

8. If you need to implement a solution where virtual machines are grouped based on their functions, allowing specific security rules for each group, what Azure service should you use?

- Application Security Groups

9. What is the role of a demilitarized zone (DMZ) when using network security groups in Azure?

- To act as a buffer between internal resources and external traffic
