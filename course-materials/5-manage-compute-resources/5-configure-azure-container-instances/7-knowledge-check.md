1. Which deployment method is used for deploying multiple containers in Azure Container Instances?

- Using an orchestrator such as Azure Kubernetes Service.

2. You are tasked to ensure two containers within a group can communicate over shared ports. Which Azure feature allows this configuration?

- Azure Container Groups

3. Which of the following is true regarding the isolation capabilities of containers compared to virtual machines?

- Containers provide lightweight isolation, but virtual machines offer stronger security boundaries.

4. Your organization requires a solution that can quickly deploy isolated applications with minimal resource usage. Which technology would best meet these requirements?

- Azure Container Instances

5. What is the first step to deploy a containerized application using Azure Container Instances?

- Create a container image using Docker.

6. Your organization needs to deploy applications that require strong security boundaries. Which technology should you choose, containers or virtual machines?

- Virtual machines, because they provide complete isolation and strong security boundaries.

7. Your multi-container group in Azure Container Instances is not functioning as expected. What could be a potential issue?

- Resource allocation for the container group is insufficient.

8. When deploying a multi-container group in Azure Container Instances, which configuration aspect is crucial for enabling external access to one of the containers?

- Configuring a public IP and exposing the correct port on the container group.

9. Your organization needs to deploy a complex application with multiple components requiring strong security boundaries. Which technology is most suitable for this deployment?

- Virtual Machines
