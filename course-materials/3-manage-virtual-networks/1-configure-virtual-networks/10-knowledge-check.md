1. Which type of IP address should be used for an Azure resource that needs to connect with Azure public-facing services?

- Public IP address

2. Your organization is planning to implement an Azure Virtual Network to connect multiple Azure resources securely. Which feature of Azure Virtual Networks allows you to create a virtual representation of your network in the cloud?

- Logical Isolation

3. What must be ensured when creating subnets within a virtual network to avoid IP address conflicts?

- The range for one subnet can't overlap with other subnet IP address ranges in the same virtual network.

4. An Azure virtual machine has lost connection to the internet after being restarted. What could be a reason for this issue related to IP addressing?

- The virtual machine uses a dynamic public IP address

5. Your company wants to use Azure Virtual Network to create logical divisions within their network for better security and performance. Which feature should you use to implement this segmentation?

- Subnets

6. Your company wants to connect its on-premises network to an Azure Virtual Network securely. Which Azure feature facilitates this type of connection?

- Site-to-Site VPN

7. Which scenario requires the use of a static IP address rather than a dynamic IP address in Azure?

- Setting up a DNS server

8. During the planning of a virtual network in Azure, what is a key consideration when defining the address space for the network?

- Ensure the address space is unique and not overlapping with on-premises networks

9. An Azure virtual machine needs to communicate with an on-premises server without exposing it to the internet. Which type of IP address should be used for the Azure virtual machine?

- Private IP address
